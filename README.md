# Mobile Controlled Home

## Overview
A smart home system controlled using a mobile phone via Bluetooth.

## Features
- Control lights
- Control fan
- Bluetooth communication
- Arduino based system

## Components
- Arduino Uno
- HC-05 Bluetooth Module
- Relay Module
- LEDs

## Software Used
- Arduino IDE
- Proteus
