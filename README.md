# Mobile Controlled Home

## Overview
A smart home system controlled using a mobile phone via Bluetooth.

## Features
- Control lights
- Control fan
- Bluetooth communication
- Arduino based system

## Components
- Arduino Uno
- HC-05 Bluetooth Module
- Relay Module
- LEDs

## Software Used
- Arduino IDE
- Proteus## Circuit Design

The circuit was designed and tested using Proteus simulation software.

## Simulation

The project functionality was verified through simulation before implementation.

Simulation files are included in the repository under the Simulation folder.
